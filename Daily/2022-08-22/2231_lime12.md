# 코드
```python
N = int(input())

for i in range(N+1):
    js = list(map(int, str(i)))
    if i + sum(js) == N:
        print(i)
        break
    if i == N:
        print(0)
```
# 풀이
- 1부터 증가하면서 자기자신에 도달할때까지 N의 생성자가 있는지 확인한다. 가장 먼저 도달하는 것이 걸리기 때문에 조건에도 만족한다.  

# 기록 
- 몇 가지 정리할 포인트.
  - 두 번째 브레이크 조건을 i+sum(js)로 걸어서 미리 정지시킬 수 있지 않는가? 이부분에서 실패를 했는데
