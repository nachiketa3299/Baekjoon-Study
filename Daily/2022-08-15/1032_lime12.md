# 1032 : 명령 프롬프트
## 소요 시간 : 24m 27s 
##소스 코드 
```python
n = int(input())
strings=[]
for i in range(n):
    strings.append(input())

patern=''

for i in range(len(strings[0])):
    cnt,j=0,0
    temp = strings[j][i]
    while j< len(strings)-1:
        j+=1
        if temp != strings[j][i]:
            patern+="?"
            cnt+=1
            break
    if cnt==0:
        patern+=temp

print(patern)
```
## 풀이
- 각 문자열 마다 같은 자리를 확인하며 하나라도 다른 값을 가진다면 패턴에 "?"을 더하고
- 모두 확인했을 때, "?"이 발견되지 않는다면 현재 문자열을 더한다.
